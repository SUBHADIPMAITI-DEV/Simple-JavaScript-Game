# Game Name: Catch the Ball

# Game Description
"Catch the Ball" is a simple, fun, and addictive game where the player controls a basket that moves left and right at the bottom of the screen. Balls fall from the top of the screen at random positions and the player's objective is to catch as many balls as possible with the basket. The game ends if a ball hits the ground.
